## 処理フロー図の記載（必要時は必須）

- 連携・非同期・状態遷移・複雑な分岐を含む仕様では、文章説明に加えて**処理フロー図**を記載する
- フロー図は `mermaid`（`flowchart` / `sequenceDiagram`）を推奨する
- 技術選定の妥当性がフロー依存の場合は、判断ログと図を相互参照できるようにする

```mermaid
sequenceDiagram
    participant U as User
    participant A as App
    participant B as Backend
    U->>A: リクエスト
    A->>B: API呼び出し
    B-->>A: レスポンス
    A-->>U: 結果表示
```
