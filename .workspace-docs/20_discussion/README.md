# discussion 運用メモ

このフォルダは、`.workspace-docs` の運用方針や文書フォーマットを検討するための議論専用領域です。  
合意前の案はここに置き、合意後に `40_plans/30_specs/50_tasks` の正本へ反映します。

---

## 1. 目的

- 合意前の論点を、正本ドキュメントと分離して管理する
- 選択肢・判断理由・決定事項を時系列で残す
- 後から「なぜその運用にしたか」を追跡できる状態にする

---

## 2. 配置ルール

- この配下には「議論中」のファイルのみを置く
- 正式な仕様・計画・タスクは置かない
- 合意済みの内容は、対応する正本へ反映後に本フォルダの記録へリンクを残す

---

## 3. 記録テンプレート

新しい議論ファイルを作るときは、以下の見出しを基本とします。

```md
# タイトル

更新日: YYYY-MM-DD
対象: （例）.workspace-docs/README.md

## 背景

## 論点

## 選択肢

## 決定

## 保留

## 正本への反映先
```

---

## 4. 反映ルール（暫定）

- まず本フォルダで議論し、決定事項を確定する
- 確定後に、正本（`40_plans/30_specs/50_tasks`）へ反映する
- 反映時は、変更対象ファイルの更新日・変更履歴も必要に応じて更新する
